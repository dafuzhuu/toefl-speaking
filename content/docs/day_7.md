## 搭建连接词框架

**表原因**

Task2 说完 proposal 的内容后解释原因，或学生提出赞同/反驳理由后的论证。笔记上有观点和原因，要怎样将两者连接起来？

原因有两种，一种是不这么做会出现什么问题，一种是这样做有什么好处

*1. 不这样做会出现什么问题*

- given that + 从句
- for fear that + 从句
- in the event that + 从句

*2. 这样做有什么好处*

- for the purpose of + 动名词
- in order to + 动词不定式
